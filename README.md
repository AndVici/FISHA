"# FISHA" 
FINDING IMAGE SIMILARITY: HISTOGRAM ANALYSIS
============================================

There are two programs that make up this project. The Matlab program **FISHAM** and the Java program **FISHA**. These programs must be run sequentially and in order.

**FISHAM**

Arguments: directory holding JPG images you wish to process.

           Directory to store output.
          
Returns: Path to output.


**FISHA**

In main you must give the output directory from **FISHAM** and the desired output directory for the results.
